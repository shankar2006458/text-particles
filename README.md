# ✨ Text Particles

A smooth interactive particle text animation built with **HTML, CSS, and JavaScript**. Type any phrase and watch thousands of particles transform into glowing text with hover repulsion effects and premium motion visuals.

## 🌐 Live Demo

🔗 [https://shankar2006458.github.io/text-particles/](https://shankar2006458.github.io/text-particles/)

## 🚀 Features

* 🎯 Real-time text particle generation
* 💖 Default heart particle shape
* ✍️ Live typing animation
* 🖱️ Hover particle burst / repel effect
* 🌌 Smooth animated background motion
* ⚡ Optimized for better low-end performance
* 📱 Responsive design for desktop & mobile
* 🎨 Premium glowing neon visuals
* 🔄 Double-click to reset animation
* ⌨️ Input length limits for smoother UX

## 🛠️ Built With

* HTML5 Canvas
* CSS3
* Vanilla JavaScript


## 📂 Installation

```bash
git clone https://github.com/shankar2006458/text-particles.git
cd text-particles
```

Then open `index.html` in your browser.
